# Padrões Editoriais — squad-conteudo

Fonte: `estilodeconteudo.md` (guia operacional completo)

## Padrão de Escrita

- Público: MEIs, empreendedores e microempreendedores brasileiros
- Tom: 80% técnico direto / 20% personalidade conversacional
- Pronome: "você" sempre (nunca "o leitor", "o usuário")
- Voz: ativa, presente

## Proibições Absolutas

### Travessão (—) no meio de frases — PROIBIDO

Nunca use travessão para quebrar frases. Alternativas:

- **Parênteses** para complemento ou nota: "O DAS (Documento de Arrecadação) é..."
- **Vírgula** para separação natural: "O DAS, um documento de arrecadação, é..."
- **Ponto + nova frase** para quebra clara: "O DAS é obrigatório. Você deve pagar todo mês."
- **Lista com bullets** para separar items: Use `-` para bullets em listas formais

Travessão só é permitido em:
- Listas de bullets (`- ` no Markdown)
- Intervalo de datas ou números: "2024–2025"

Ver `checklists/checklist-pre-publicacao.md` para lista completa.

Palavras e conectivos da Lista Negra (Regra #3) resultam em VETO imediato — sem exceções.

## H2s

Todo H2 deve ter fonte real verificável no Google (PAA, relacionadas, autocomplete ou concorrentes).
H2s genéricos são proibidos. Ver `agents/pesquisador-seo.md` para lista completa.
